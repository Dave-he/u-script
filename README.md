# Universal Script

通用脚本集合，包含各种实用的跨平台脚本工具。

## 脚本列表

### 1. Git代理设置脚本 (git_proxy.py)
跨平台的Git代理设置工具，支持：
- 检查当前Git代理设置
- 设置HTTP/SOCKS5代理
- 取消代理设置
- 交互式操作界面

#### 使用方法
```bash
python git_proxy.py
```

#### 功能特性
- 自动检测当前代理设置
- 支持HTTP和SOCKS5协议
- 支持自定义代理地址
- 默认代理地址：127.0.0.1:10808
- 跨平台兼容（Windows/macOS/Linux）

#### 详细文档
查看完整使用指南：[docs/git-proxy-usage.md](docs/git-proxy-usage.md)
